Auto-Install-GLPI

Installation automatique de GLPI : Un script Bash pour installer et configurer facilement GLPI sur un serveur Debian. Il gère l'installation d'Apache, PHP, MariaDB, et la version de GLPI souhaitée. Ce script simplifie grandement le processus d'installation, le rendant accessible même aux utilisateurs moins expérimentés.

Auto-Update-GLPI

Ce script automatise le processus de mise à jour de GLPI à la version souhaitée. Il gère la sauvegarde des composants essentiels (Base de données) et l'installation vers la version sélectionnée. Cet outil est particulièrement utile pour les administrateurs système qui doivent maintenir GLPI à jour régulièrement.

Auto-Update-Centreon

Ce script Bash permet de mettre à jour automatiquement Centreon vers une version souhaitée. Il offre également une option pour sauvegarder la base de données avant la mise à jour. Cet outil est essentiel pour les équipes IT qui utilisent Centreon pour la surveillance de leur infrastructure.
